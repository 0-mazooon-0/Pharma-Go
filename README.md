Pharma-Go is a very simple pharmacy application that facilitates searching and buying medicines for the customers. Pharmacists can also manage the pharmacy stock easily from the app.

These are the functionalities a customer has:

Searching medicines,
Reviewing the price and description of this medicine,
Cart to add or remove medicines,
Uploading a prescription as a PDF,
Pay and checkout

The pharmacist's also can manage stock by: 
Adding
Removing medicines
Update the price of an existing medicine.
